Please fork this repository and build a service that provides movie information. Create three pages: a home page, a category detail page, and a movie detail page.

On the home page, arrange movie thumbnails by category and include links to the category detail pages for each category. Also, ensure that clicking on a movie thumbnail takes the user to the detail page.

The category detail page should display a list of movie thumbnails. As with the home page, clicking a movie thumbnail should redirect the user to the movie detail page.

The movie detail page should display the movie title, description, and number of likes, and show a “Comments” list in the right sidebar.

Please use the following GraphQL queries (the query definitions are already located in `lib/graphql/query`).

For more details or to view other queries, please refer to https://develop.api.samansa.com/graphiql.

    getHomeScreens
        Returns the movie categories and their respective lists to be displayed on the movie list page
    getCategory
        Returns a list of movies contained in a specified movie category ID
    getOriginalVideo
        Returns detailed information for a specified movie ID
    getVideoComments
        Returns a list of comments for a specified movie ID

Practical optimizations are highly encouraged! (For example, using the first getHomeScreens method to retrieve only the categories, and then fetching the movies for each category later)
